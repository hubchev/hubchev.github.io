4 Net Present Value Exercise

You are considering investing in a project that costs 50,000 today and is expected to generate cash flows of 20,000 per year for the next 5 years. You have a required rate of return of 8%. Calculate the NPV of the project and decide whether to invest in it or not.

Assuming that the cash flows occur at the end of each year, we can use the following formula to calculate the NPV of the project:

NPV = CF1 / (1 + r)^1 + CF2 / (1 + r)^2 + … + CFn / (1 + r)^n - Initial Investment

where CF is the cash flow for each year, r is the required rate of return, n is the number of years, and the Initial Investment is the cost of the project.

Plugging in the values from the exercise, we get:

NPV = -50,000 + 20,000 / (1 + 0.08)^1 + 20,000 / (1 + 0.08)^2 + 20,000 / (1 + 0.08)^3 + 20,000 / (1 + 0.08)^4 + 20,000 / (1 + 0.08)^5

NPV = -50,000 + 18,518.52 + 17,131.40 + 15,813.94 + 14,560.51 + 13,366.61

NPV = 28,330.98

Since the NPV is positive, we can conclude that the project is expected to generate a return that is higher than the required rate of return. Therefore, we should invest in the project.
